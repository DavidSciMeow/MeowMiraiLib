语音信息(构造参数任选其一，出现多个参数时，按照voiceId > url > path > base64的优先级)

参数
voiceId语音的voiceId，不为空时将忽略url属性
url语音的URL，发送时可作网络语音的链接；接收时为腾讯语音服务器的链接，可用于语音下载
path语音的路径，发送本地语音，路径相对于 JVM 工作路径（默认是当前路径，可通过 -Duser.dir=...指定），也可传入绝对路径。
base64语音的 Base64 编码
length返回的语音长度, 发送消息时可以不传
